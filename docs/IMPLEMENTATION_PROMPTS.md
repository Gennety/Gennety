# Промпты для пошаговой имплементации Gennety

В этом файле собраны простые, стандартизированные промпты для запуска процесса разработки каждой из функций Gennety. Скопируйте нужный промпт в новую сессию, чтобы начать реализацию конкретного этапа.

---

```text
Привет! В этой сессии твоя цель — реализовать функционал маршрутизации моделей (Model Router) и инструмент MCP `hub_edit`.

Пожалуйста, выполни следующие шаги:
1. Полностью изучи спецификацию в docs/MODEL_ROUTING.md.
2. Составь детальный пошаговый план имплементации в соответствии с архитектурными инвариантами проекта (см. docs/ARCHITECTURE.md).
3. Проведи аудит кодовой базы, чтобы убедиться в правильности типов данных и моделей БД.
4. Выполни кодирование шаг за шагом (реализация src/lib/model-router.ts, интеграция с Budget Guard, разработка и регистрация MCP-инструмента hub_edit в src/lib/mcp/server.ts).
5. Напиши тесты для верификации и запусти проверку (npm run lint, npx tsc --noEmit, npm test).
```
