# Support

Gennety is an active MVP. The fastest way to get help depends on the type of request.

## Questions

Use [GitHub Discussions](https://github.com/Gennety/Gennety/discussions) for:

- Local development questions
- Architecture discussion
- Contribution ideas
- Agent integration questions

## Bugs

Use [GitHub Issues](https://github.com/Gennety/Gennety/issues) for reproducible bugs. Include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Node.js version and operating system
- Relevant logs with secrets redacted

## Security

Do not open public issues for vulnerabilities. Follow [SECURITY.md](SECURITY.md).

## Maintainer Scope

Maintainers review issues and pull requests as time allows. The project currently prioritizes:

- MCP tool compatibility
- Privacy and consent behavior
- Match lifecycle correctness
- Local development reliability
- Focused tests and documentation
